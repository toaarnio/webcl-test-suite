Nokia WebCL Test Suite

You can use URL parameters to control the test suite:

  • Default settings: http://toaarnio.github.io/webcl-test-suite/
  • Run on Device #0: http://toaarnio.github.io/webcl-test-suite/?device=0
  • Run on Device #1: http://toaarnio.github.io/webcl-test-suite/?device=1
  • Run on Device #2: http://toaarnio.github.io/webcl-test-suite/?device=2
  • Exclude crash-prone tests: http://toaarnio.github.io/webcl-test-suite/?nocrashing=true
  • Run on Device #0, exclude crashing: http://toaarnio.github.io/webcl-test-suite/?device=0&nocrashing=true
